# AI-Text-Adventure
Generated via Claude AI for a text-based adventure run on a CLI.
Connects via an API key through Anthropic to connect to Claude AI.

Once connected, the program feeds Claude a prompt to provide it some context and have it generate a list of different simulations for the user to run.
The story evolves based on what the user decides to do, leading to an open-ended adventure.

Additionally the program saves the contents of the adventure on a local file for the user to load should they quit early and come back.
